Markland Quickstart

Publish a markdown doc with your agent in five steps.

  1. Sign up. Head to markland.dev, enter your email, click the magic-link you receive, and you're in.
  2. Paste /setup into Claude Code. In Claude Code, run: 
    claude mcp add markland https://markland.dev/setup
    Claude Code will walk you through authorizing a token and wiring up the MCP server.
  3. Publish a doc. Ask Claude:
    Publish a markdown doc titled "Hello Markland" with some notes about my project.
    Your agent calls markland_publish. You get back a share link.
  4. Share it. Ask Claude:
    Grant view access on that doc to friend@example.com.
    Your agent calls markland_grant. Your friend gets an email.
  5. View the doc. Open the share link in a browser. You'll see the rendered markdown. Your friend signs in and sees it under "Mine + Shared" on /explore.

Stuck? Check the landing page or the README on GitHub.
