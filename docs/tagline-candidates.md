# Tagline Candidates

A running list of tagline candidates to test. Generated during positioning iteration on 2026-04-18.

## Current selection

**Shared notes for you and your agents.**

Reasoning: warmest option in the collaboration set. "You and your agents" is intimate, not institutional. Reads as a product description rather than a category definition. Works for readers who don't know what MCP is.

## Bench (ready to test)

### Collaboration angle — iterations on "Shared memory for humans and agents"

Varies the noun and the warmth. No autonomy language.

| Tagline | Angle |
|---|---|
| Shared memory for humans and agents. | Baseline — evocative, dual meaning of "memory" |
| Shared notes for you and your agents. | **Current selection** — warmest, intimate |
| Shared context for humans and agents. | LLM vocab — "context" reads as on-theme to devs |
| A notebook humans and agents share. | Concrete object, "share" as verb |
| Shared pages for humans and agents. | Markdown-native, tangible |
| A shared workspace for humans and agents. | Professional — Notion/Slack own this word |
| The knowledge layer humans and agents edit together. | Explicit about co-editing, B2B register |
| Docs humans and agents edit together. | Plainest, hard to dislike/remember |
| One notebook for you and your agents. | Intimate, "one" sells the shared-surface idea |
| The place humans and agents keep notes together. | Warm, human — undersells M2M |

## Archived for later use (not for hero)

- **Git is overkill. Google Docs isn't listening.** — Gold for about-page hero or manifesto section. Needs a completing line ("Markland is built for agents."). Wasted as a one-liner.

## Notes for testing

- "Without you" and "no permissions" framing is out — not the idea. Markland is about collaboration, not agent-runs-wild autonomy.
- Register varies across the bench: #2 for general audience, #3 for dev-savvy (uses "context"), #1 for visionary.
- Consider hero + subhead pairings where the hero is a sharper line (e.g. "Every agent gets an edit button.") and the subhead does the category work ("Shared memory for humans and agents.").
