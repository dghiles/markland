# Agents — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

> **Note:** No git commits during execution — this project is not a git repo at time of writing; the user manages version control manually. Where commit steps would normally appear, a "verification" step is substituted.

**Goal:** Make agents first-class principals. A user can create agents, mint agent tokens (`mk_agt_<random>`), and grant docs directly to an agent by id. User-owned agents inherit their owner's grants via the permission-resolver. Agents publishing docs attribute ownership to their owning user. The data model already supports service-owned agents (admins can create them via a helper script) but no user-facing surface ships in this plan.

**Architecture:** A new `agents` table holds agent identity. A new `service/agents.py` holds create/list/revoke. `service/auth.py` gains `create_agent_token` and a branch in `resolve_token` that returns a `Principal` whose `user_id` is the agent's owning user (user-owned) or `None` (service-owned), and `is_admin=False`. `service/docs.py check_permission` wires the §5 step-3 inheritance rule: a user-owned-agent principal with no direct grant inherits the grant on `(doc, agent.owner_id, 'user')`. `service/grants.py grant()` now accepts an email (→ user) or an `agt_…` id (→ agent). HTTP routes expose `/api/agents` and `/api/agents/{id}/tokens`; an HTML `/settings/agents` page wraps them. An `EmailClient` trigger notifies agent owners when their agent is granted access. A `scripts/create_service_agent.py` helper exists for admin-only service-agent creation.

**Tech Stack:** Python 3.12, FastAPI, FastMCP, SQLite (WAL), argon2-cffi, jinja2, pytest. Existing Markland stack; no new external dependencies.

**Scope excluded (this plan):**
- No service-agent web UI (helper script only).
- No admin promotion UI.
- No OAuth / service-agent registration surface.
- No agent-to-agent grant delegation.
- No agent rate limits beyond whatever lands in plan 10.
- No `markland_grant` tool changes for service-agent mail semantics beyond "send mail iff agent is user-owned."

            display_name TEXT NOT NULL,

```python
"""Agent CRUD — user-owned and service-owned agents."""

from __future__ import annotations

import sqlite3

from markland.models import Agent


def _row_to_agent(row: sqlite3.Row) -> Agent:
    return Agent(
        id=row["id"],
        display_name=row["display_name"],
        owner_type=row["owner_type"],
        owner_id=row["owner_id"],
        created_at=row["created_at"],
        revoked_at=row["revoked_at"],
    )


def get_agent(conn: sqlite3.Connection, agent_id: str) -> Agent | None:
    """Return the agent row (active or revoked), or None if missing."""
    row = conn.execute("SELECT * FROM agents WHERE id = ?", (agent_id,)).fetchone()
    return _row_to_agent(row) if row else None


def create_agent(
    conn: sqlite3.Connection,
    owner_user_id: str,
    display_name: str,
) -> Agent:
    """Create a user-owned agent. Raises ValueError if validation fails."""
    name = (display_name or "").strip()
    if not name:
        raise ValueError("display_name_required")

    user = conn.execute("SELECT id FROM users WHERE id = ?", (owner_user_id,)).fetchone()
    if user is None:
        raise ValueError("user_not_found")

    agent = Agent(
        id=Agent.generate_id(),
        display_name=name,
        owner_type="user",
        owner_id=owner_user_id,
        created_at=Agent.now(),
        revoked_at=None,
    )
    conn.execute(
        "INSERT INTO agents(id, display_name, owner_type, owner_id, created_at, revoked_at) "
        "VALUES (?, ?, 'user', ?, ?, NULL)",
        (agent.id, agent.display_name, agent.owner_id, agent.created_at),
    )
    conn.commit()
    return agent


def create_service_agent(
    conn: sqlite3.Connection,
    service_id: str,
    display_name: str,
) -> Agent:
    """Create a service-owned agent. Admin-only at the caller level; no web UI."""
    name = (display_name or "").strip()
    if not name:
        raise ValueError("display_name_required")
    if not service_id or not service_id.startswith("svc_"):
        raise ValueError("service_id_required")

    agent = Agent(
        id=Agent.generate_id(),
        display_name=name,
        owner_type="service",
        owner_id=service_id,
        created_at=Agent.now(),
        revoked_at=None,
    )
    conn.execute(
        "INSERT INTO agents(id, display_name, owner_type, owner_id, created_at, revoked_at) "
        "VALUES (?, ?, 'service', ?, ?, NULL)",
        (agent.id, agent.display_name, agent.owner_id, agent.created_at),
    )
    conn.commit()
    return agent


def list_agents(conn: sqlite3.Connection, owner_user_id: str) -> list[Agent]:
    """List active (non-revoked) user-owned agents for a given user."""
    rows = conn.execute(
        "SELECT * FROM agents "
        "WHERE owner_type = 'user' AND owner_id = ? AND revoked_at IS NULL "
        "ORDER BY created_at DESC",
        (owner_user_id,),
    ).fetchall()
    return [_row_to_agent(r) for r in rows]


def revoke_agent(
    conn: sqlite3.Connection,
    agent_id: str,
    owner_user_id: str,
) -> None:
    """Soft-delete an agent by setting revoked_at. Owner-only."""
    row = conn.execute("SELECT owner_type, owner_id FROM agents WHERE id = ?", (agent_id,)).fetchone()
    if row is None:
        raise LookupError(f"agent_not_found: {agent_id}")
    if row["owner_type"] != "user" or row["owner_id"] != owner_user_id:
        raise PermissionError("not_agent_owner")

    conn.execute(
        "UPDATE agents SET revoked_at = ? WHERE id = ? AND revoked_at IS NULL",
        (Agent.now(), agent_id),
    )
    conn.commit()
```

## Task 5: Wire inheritance in `check_permission`

**Files:**
- Modify: `src/markland/service/docs.py`
- Create: `tests/test_docs_agent_inheritance.py`

The §5 resolver is:
1. Owner → allow anything.
2. Direct grant `(doc, principal)` → use its level.
3. **User-owned agent + grant `(doc, agent.owner)` → inherit.** ← this task.

import logging

from markland.service.email import EmailSendError

logger = logging.getLogger("markland.grants")


def _maybe_send_agent_grant_email(
    conn, *, email_client, doc_id, agent_row, level, granted_by,
):
    """Email the user-owning principal of an agent on a grant.

    Caller has already verified `agent_row["owner_type"] == "user"`.
    `granted_by` is the actor Principal passed into `grant()` (has
    `display_name` and `principal_id`).
    """
    owner_row = conn.execute(
        "SELECT email, display_name FROM users WHERE id = ?",
        (agent_row["owner_id"],),  # DB column — stays `owner_id`
    ).fetchone()
    if owner_row is None:
        return

    doc_row = conn.execute(
        "SELECT title FROM documents WHERE id = ?", (doc_id,),
    ).fetchone()
    doc_title = doc_row["title"] if doc_row else doc_id

    subject = f"{granted_by.display_name} granted your agent {level} access to {doc_title}"
    html = (
        f"<p>{granted_by.display_name} granted your agent "
        f"<strong>{agent_row['display_name']}</strong> "
        f"<strong>{level}</strong> access to <em>{doc_title}</em>.</p>"
    )

    try:
        email_client.send(to=owner_row["email"], subject=subject, html=html)
    except EmailSendError as exc:
        logger.warning("agent-grant email failed: %s", exc)

```python
"""HTTP routes for agent management under /api/agents."""

from __future__ import annotations

from fastapi import APIRouter, Depends, HTTPException, Request, status
from pydantic import BaseModel, Field

from markland.service import agents as agents_svc
from markland.service import auth as auth_svc


class CreateAgentBody(BaseModel):
    display_name: str = Field(min_length=1, max_length=200)


class CreateAgentTokenBody(BaseModel):
    label: str = Field(min_length=1, max_length=200)


def build_agents_router(db_conn) -> APIRouter:
    router = APIRouter(prefix="/api/agents", tags=["agents"])

    def _session_user(request: Request):
        # Plan 2 provides resolve_session — returns the logged-in User or 401.
        user = auth_svc.resolve_session(request, db_conn)
        if user is None:
            raise HTTPException(status_code=401, detail="unauthenticated")
        return user

    @router.get("", response_model=list[dict])
    def list_agents_endpoint(user=Depends(_session_user)):
        agents = agents_svc.list_agents(db_conn, owner_user_id=user.id)
        return [
            {
                "id": a.id,
                "display_name": a.display_name,
                "owner_type": a.owner_type,
                "owner_id": a.owner_id,
                "created_at": a.created_at,
            }
            for a in agents
        ]

    @router.post("", status_code=status.HTTP_201_CREATED)
    def create_agent_endpoint(body: CreateAgentBody, user=Depends(_session_user)):
        try:
            agent = agents_svc.create_agent(
                db_conn, owner_user_id=user.id, display_name=body.display_name,
            )
        except ValueError as exc:
            raise HTTPException(status_code=400, detail=str(exc)) from exc
        return {
            "id": agent.id,
            "display_name": agent.display_name,
            "owner_type": agent.owner_type,
            "owner_id": agent.owner_id,
            "created_at": agent.created_at,
        }

    @router.delete("/{agent_id}", status_code=status.HTTP_204_NO_CONTENT)
    def revoke_agent_endpoint(agent_id: str, user=Depends(_session_user)):
        try:
            agents_svc.revoke_agent(db_conn, agent_id, owner_user_id=user.id)
        except LookupError:
            raise HTTPException(status_code=404, detail="not_found")
        except PermissionError:
            raise HTTPException(status_code=403, detail="forbidden")
        return None

    @router.post("/{agent_id}/tokens", status_code=status.HTTP_201_CREATED)
    def create_agent_token_endpoint(
        agent_id: str,
        body: CreateAgentTokenBody,
        user=Depends(_session_user),
    ):
        try:
            tok_id, plaintext = auth_svc.create_agent_token(
                db_conn, agent_id=agent_id, owner_user_id=user.id, label=body.label,
            )
        except LookupError:
            raise HTTPException(status_code=404, detail="not_found")
        except PermissionError:
            raise HTTPException(status_code=403, detail="forbidden")
        except ValueError as exc:
            raise HTTPException(status_code=400, detail=str(exc)) from exc
        return {"id": tok_id, "plaintext": plaintext, "label": body.label}

    @router.delete(
        "/{agent_id}/tokens/{token_id}", status_code=status.HTTP_204_NO_CONTENT,
    )
    def revoke_agent_token_endpoint(
        agent_id: str, token_id: str, user=Depends(_session_user),
    ):
        # Token must belong to an agent owned by this user.
        row = db_conn.execute(
            "SELECT t.id FROM tokens t "
            "JOIN agents a ON a.id = t.principal_id "
            "WHERE t.id = ? AND t.principal_type = 'agent' "
            "  AND a.owner_type = 'user' AND a.owner_id = ? AND a.id = ?",
            (token_id, user.id, agent_id),
        ).fetchone()
        if row is None:
            raise HTTPException(status_code=404, detail="not_found")
        db_conn.execute(
            "UPDATE tokens SET revoked_at = ? WHERE id = ?",
            (auth_svc.now_iso(), token_id),
        )
        db_conn.commit()
        return None

    return router
```

import pytest

from markland.db import init_db
from markland.service import agents as agents_svc
from markland.service import auth as auth_svc
from markland.service import docs as docs_svc
from markland.service import grants as grants_svc
from markland.service.errors import NotFound, PermissionDenied


class _UserPrincipal:
    def __init__(self, uid, name="U"):
        self.principal_type = "user"
        self.principal_id = uid
        self.user_id = uid
        self.display_name = name
        self.is_admin = False


@pytest.fixture
def conn(tmp_path):
    c = init_db(tmp_path / "smoke.db")
    for uid, email, name in [
        ("usr_alice", "alice@x", "Alice"),
        ("usr_bob", "bob@x", "Bob"),
    ]:
        c.execute(
            "INSERT INTO users(id, email, display_name, is_admin, created_at) "
            "VALUES (?, ?, ?, 0, ?)",
            (uid, email, name, "2026-04-19T00:00:00+00:00"),
        )
    c.commit()
    yield c
    c.close()


def test_full_agent_inheritance_flow(conn):
    # 1. Alice publishes a doc.
    alice = _UserPrincipal("usr_alice", "Alice")
    doc = docs_svc.publish_doc(
        conn, base_url="http://t", title="Alice's Notes", content="# Hello",
        principal=alice,
    )

    # 2. Bob creates an agent and mints an agent token.
    bob = _UserPrincipal("usr_bob", "Bob")
    agent = agents_svc.create_agent(conn, "usr_bob", "bob-scribe")
    _, plaintext = auth_svc.create_agent_token(
        conn, agent_id=agent.id, owner_user_id="usr_bob", label="laptop",
    )
    assert plaintext.startswith("mk_agt_")

    # 3. Resolving the agent token yields a Principal tied to Bob.
    principal = auth_svc.resolve_token(conn, plaintext)
    assert principal is not None
    assert principal.principal_type == "agent"
    assert principal.user_id == "usr_bob"

    # 4. Without a grant, the agent cannot see Alice's doc.
    with pytest.raises(NotFound):
        docs_svc.check_permission(
            conn, doc_id=doc["id"], principal=principal, action="view",
        )

    # 5. Alice grants Bob view access.
    grants_svc.grant(
        conn, base_url="http://t", principal=alice,
        doc_id=doc["id"], target="bob@x", level="view", email_client=None,
    )

    # 6. Bob's agent now inherits the grant.
    tag = docs_svc.check_permission(
        conn, doc_id=doc["id"], principal=principal, action="view",
    )
    assert tag == "view"
    # 7. View-only inheritance: edit is still denied.
    with pytest.raises(PermissionDenied):
        docs_svc.check_permission(
            conn, doc_id=doc["id"], principal=principal, action="edit",
        )

    # 8. Alice upgrades Bob to edit; the agent inherits edit.
    grants_svc.grant(
        conn, base_url="http://t", principal=alice,
        doc_id=doc["id"], target="bob@x", level="edit", email_client=None,
    )
    tag = docs_svc.check_permission(
        conn, doc_id=doc["id"], principal=principal, action="edit",
    )
    assert tag == "edit"

    # 9. Revoking the agent invalidates its token.
    agents_svc.revoke_agent(conn, agent.id, owner_user_id="usr_bob")
    assert auth_svc.resolve_token(conn, plaintext) is None

- `/settings/agents` renders for logged-in users and shows existing agents; plaintext is shown once after token creation.
- `scripts/create_service_agent.py` creates a service agent and prints its token exactly once.
- The end-to-end smoke test demonstrates inheritance working for userA-grants-userB → Bob's agent.

## What this plan does NOT deliver

- No web UI for service-owned agents. The helper script (`scripts/create_service_agent.py`) is the only creation path.
- No admin UI for any agent management — no browser surface to promote a user to admin, list all service agents, or rotate service-agent tokens.
- No OAuth / service-registration flow. Deferred until a real integration partner materializes (spec §3, §15).
- No invite-link fallback when granting to an email with no existing user account (plan 5 delivers this).
- No device-flow onboarding for agent tokens; humans paste tokens manually for now (plan 6 delivers device flow for user tokens).
- No audit-log entries for agent create/revoke/grant-inheritance (plan 10).
- No per-agent rate-limiting policy (plan 10).
- No `markland_grant` tool copy changes specific to agent targets — the tool is delegated to `service/grants.grant` which now handles `agt_…` ids transparently.
